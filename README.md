# mind_reader
Microsoft xiaoice mind reader interface encapsulation （python2.7）<br>
150 行代码实现微软小冰读心术 http 接口封装（python2.7），命令行交互。

# Quick start

### 1、clone project

```
$ git clone https://github.com/ZealiGit/mind_reader.git
```

### 2、environment
```
$ cd mind_reader
$ pip install -r requirements.txt
```

### 3、run

```
$ python index.py
```
